# Student Management System

A Full Stack Java project using Core Java, JDBC, MySQL with MVC Architecture.

## Technologies Used
- Core Java, JDBC, MySQL 8.0
- HTML5, CSS3, JavaScript (ES6+)
- MVC Architecture

## Project Structure
- `src/model/` - Student.java (Model)
- `src/dao/` - StudentDAO.java (CRUD operations)
- `src/util/` - DBConnection.java (DB connection)
- `src/main/` - Main.java (Entry point)
- `web/` - Frontend (HTML, CSS, JS)
- `database/` - schema.sql

## Features
- Add, View, Search, Update, Delete students
- PreparedStatements (SQL Injection prevention)
- Exception Handling with try-with-resources
- Responsive Frontend UI

## How to Run
1. MySQL lo `database/schema.sql` run cheyyi
2. `DBConnection.java` lo password update cheyyi
3. `Main.java` run cheyyi
